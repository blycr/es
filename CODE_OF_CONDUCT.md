# 行为准则

## 基本要求

本项目希望讨论保持尊重、直接，并聚焦于改进软件本身。

参与者应当：

- 使用清晰、尊重的表达方式
- 在技术分歧中默认对方出于善意
- 提供具体、可执行的反馈
- 避免人身攻击、侮辱性表达或轻蔑态度

## 不被接受的行为

以下行为不被接受：

- 人身攻击或贬损性语言
- 在 Issue 或 Pull Request 中持续制造干扰
- 未经允许公开他人隐私信息
- 在项目相关空间中实施骚扰行为

## 维护者职责

维护者可以对不符合本准则的评论、Issue、Pull Request 或其他贡献进行编辑、隐藏、锁定或移除。

## 反馈方式

如果你遇到或观察到不当行为，请私下联系维护者。

涉及安全问题时，请按 [SECURITY.md](./SECURITY.md) 中的方式反馈。
